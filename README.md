# Weather App - 5 day weather forecast
A simple application to display 5-day weather forecast using the OpenWeatherMap API. 
### Getting Started

## Pre-requisites
* Node.js 9.8.0 and above

## Run
```
git clone
cd weatherApp
npm i
```


## Start the dev server
```

npm start

```

### Tech Stack

* React.js
* Redux
* JavaScript (ES6)
* HTML5
* Bootstrap
